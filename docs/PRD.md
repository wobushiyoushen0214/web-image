# Web Image · 产品需求文档（PRD v0.1）

> 状态：Phase 0 定稿 · 2026-05-09
> 待 review：品牌名、积分定价数字（接入真实上游成本后校准）

---

## 1. 一句话定位

让普通人也能做出设计师级别图的 AI 生图站，主打**风格化模板（Skills）**——
不用懂 prompt，也能复用大神的风格出图。

---

## 2. 市场与渠道

- **主市场**：中国大陆（先国内验证模型，海外作为 v2/v3 计划）
- **冷启动渠道**：小红书 + 抖音（图文/短视频种草） + 微信群裂变
- **货币**：人民币
- **合规**：ICP 备案、内容审核（prompt + 出图双拦截）、用户实名（手机号）

---

## 3. 用户画像

| 维度 | C 端兴趣用户（80% 流量） | 创作者/设计师（20% 流量、60% 营收） |
|---|---|---|
| 场景 | 头像、表情包、装 X、给娃做卡通照 | 商单素材、风格探索、二创封面 |
| 核心诉求 | 简单 + 好玩 + 能炫 | 可控 + 一致性 + 高分辨率 |
| 付费意愿 | 5–30 元尝鲜 | 50–300 元/月持续付费 |
| UI 倾向 | 大图、模板墙、一键套用 | 参数全开、批量、图层 |
| 留存抓手 | 模板更新、社交分享 | Skills 收藏、批量、高清 |

**设计原则**：默认 UI 给 C 端，设计师走"专业模式"开关解锁全参数。**不做两个 App。**

---

## 4. 商业模式：纯积分制

**积分单位**：1 积分 ≈ ¥0.1（充值时打折）

### 4.1 消耗表（起步参数，待真实成本校准）

| 操作 | 消耗积分 |
|---|---|
| 文生图 1024² 标准 | 5 |
| 文生图 1024² 高质量 | 10 |
| 文生图 1536² | 12 |
| 图生图（编辑） | 8 |
| Prompt 美化（enhance） | 1 |
| 高清放大（v2） | 5 |

### 4.2 充值档位

| 档位 | 价格 | 积分 | 折算 | 备注 |
|---|---|---|---|---|
| 体验包 | ¥9.9 | 100 | ¥0.099 | 新用户首充 ¥6.6 |
| 标准包 | ¥39 | 500 | ¥0.078 | **主推** |
| 大师包 | ¥99 | 1500 | ¥0.066 | 创作者档 |
| 月度无忧 | ¥199 | 5000 + 不限美化 | ¥0.04 | 准订阅 |

### 4.3 福利

- 注册送 30 积分（够 6 张标准图试水）
- 邀请好友：双方各得 50 积分
- 每日签到：1 积分（防流失）

### 4.4 成本红线

毛利目标 ≥ 60%。每张图真实上游成本接入后必须校准定价。

## 5. MVP 范围（Phase 1 必做）

| # | 模块 | 备注 |
|---|---|---|
| 1 | 用户系统 | 手机号 + 微信扫码 |
| 2 | 积分账户 | 充值/消费/赠送/流水 |
| 3 | 支付 | 微信支付 + 支付宝 |
| 4 | 云端历史 + 图片存储 | 替换 localStorage |
| 5 | 内容审核 | 入参 + 出图双拦截 |
| 6 | 落地页 + 工作台分离 | `/` 营销，`/app` 工作台 |
| 7 | 法务文档 | ToS / 隐私 / 退款 / ICP |

**不做**：模板社区、多模型对比、局部重绘、扩图、团队、API、海外支付、国际化（全部移到 Phase 2+）

## 7. 品牌（待定）

当前名 `Web Image` 通用、不可搜、不可记，**必须改名**。

候选方向：
- **风格化**：Skillify / StylePic / Pictune / Promptly
- **中文调性**：图灵造像 / 画几张 / 图想
- **短记忆**：pict.ai / gen.do / pix.fun

域名建议：`.com` + `.cn`（备案）+ `.ai`（备份）。

## 8. 路线图概览

| 阶段 | 周期 | 核心目标 |
|---|---|---|
| **Phase 0** | 已完成 | 定位、用户、定价、范围对齐 |
